# AI Sentinel

> Open-source AI-generated content detection for images and videos.

AI Sentinel analyzes media files using multiple forensic techniques to estimate the probability that content was generated by artificial intelligence. It provides a confidence-scored assessment with detailed breakdowns per analysis method.

## Features

- **Multi-method image analysis** — Error Level Analysis (ELA), frequency domain (FFT), statistical analysis, metadata inspection, and texture analysis (LBP)
- **Video detection** — Frame sampling with temporal consistency and optical flow motion analysis
- **Web interface** — Clean drag-and-drop UI with animated results visualization
- **REST API** — Simple JSON API for programmatic access
- **Privacy-first** — All analysis runs locally. No files are sent to external services.

## Quick Start

### Prerequisites

- Python 3.10 or higher
- pip

### Installation

```bash
# Clone the repository
git clone https://github.com/digitalmoveio/ai-sentinel.git
cd ai-sentinel

# Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Run

```bash
python -m app.main
```

Open [http://localhost:5000](http://localhost:5000) in your browser.

## How It Works

AI Sentinel combines five independent analysis methods for images. Each method examines a different aspect of the media and produces an independent score. These scores are combined using weighted averaging to produce a final AI probability.

### Detection Methods

**Error Level Analysis (ELA)** — Re-compresses the image at a known quality and measures the difference.

**Frequency Domain Analysis (FFT)** — Transforms the image into the frequency domain to detect AI artifacts.

**Statistical Analysis** — Examines pixel distributions, noise levels, and color uniqueness.

**Metadata Analysis** — Inspects EXIF data for camera information and software signatures.

**Texture Analysis (LBP)** — Uses Local Binary Patterns to assess texture entropy and consistency.

### Video Analysis

For videos, AI Sentinel samples key frames and runs image analysis on each, plus temporal consistency and motion analysis.

## API

### POST /api/analyze

Upload a file for analysis.

```bash
curl -X POST -F "file=@photo.jpg" http://localhost:5000/api/analyze
```

### GET /api/formats

Returns supported file formats.

### GET /api/health

Health check endpoint.
